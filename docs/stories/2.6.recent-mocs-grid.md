# Story 2.6: Recent MOCs Grid

## Status

Approved

## Story

**As a** user,
**I want** to see my recent MOCs with thumbnails,
**so that** I can quickly access them.

## Acceptance Criteria

1. ⬜ RecentMocsGrid component created
2. ⬜ Displays up to 5 MOC cards
3. ⬜ Each card shows: thumbnail, title, date
4. ⬜ Clicking card navigates to MOC detail
5. ⬜ Responsive horizontal scroll on mobile

## Dev Notes

### Component Structure

```typescript
interface RecentMocsGridProps {
  mocs: RecentMoc[]
}

export function RecentMocsGrid({ mocs }: RecentMocsGridProps) {
  return (
    <section>
      <h2 className="text-xl font-semibold mb-4">Recent MOCs</h2>
      <div className="flex gap-4 overflow-x-auto pb-2 md:grid md:grid-cols-5">
        {mocs.map((moc) => (
          <Link key={moc.id} to={`/gallery/${moc.id}`}>
            <Card className="w-40 md:w-auto flex-shrink-0">
              <img src={moc.thumbnail} alt={moc.title} />
              <p className="font-medium truncate">{moc.title}</p>
              <p className="text-sm text-muted-foreground">
                {formatRelativeDate(moc.createdAt)}
              </p>
            </Card>
          </Link>
        ))}
      </div>
    </section>
  )
}
```
