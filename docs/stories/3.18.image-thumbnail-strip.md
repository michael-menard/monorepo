# Story 3.18: Image Thumbnail Strip

## Status

Approved

## Story

**As a** user,
**I want** to see thumbnails of all MOC images,
**so that** I can select which image to view.

## Acceptance Criteria

1. ⬜ ThumbnailStrip component created
2. ⬜ Horizontal row of thumbnails
3. ⬜ Click to select image
4. ⬜ Active thumbnail highlighted
5. ⬜ Horizontal scroll if many images

## Dev Notes

```typescript
interface ThumbnailStripProps {
  images: MocImage[]
  selectedIndex: number
  onSelect: (index: number) => void
}

export function ThumbnailStrip({ images, selectedIndex, onSelect }: ThumbnailStripProps) {
  return (
    <div className="flex gap-2 overflow-x-auto pb-2">
      {images.map((image, index) => (
        <button
          key={image.id}
          onClick={() => onSelect(index)}
          className={cn(
            'flex-shrink-0 w-16 h-16 rounded-md overflow-hidden border-2',
            selectedIndex === index ? 'border-primary' : 'border-transparent'
          )}
        >
          <img
            src={image.thumbnailUrl}
            alt={`Thumbnail ${index + 1}`}
            className="w-full h-full object-cover"
          />
        </button>
      ))}
    </div>
  )
}
```
