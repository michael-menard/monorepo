## Story

**As a** user,
**I want** MOCs displayed in a responsive grid,
**so that** I can see many MOCs at once.

## Acceptance Criteria

1. ⬜ CollectionGrid component created
2. ⬜ Responsive: 1/2/3/4 columns
3. ⬜ Renders MocCard for each item
4. ⬜ Consistent gap between cards
5. ⬜ Handles empty array

## Dev Notes

```typescript
interface CollectionGridProps {
  mocs: MocSummary[]
}

export function CollectionGrid({ mocs }: CollectionGridProps) {
  if (mocs.length === 0) {
    return <CollectionEmptyState />
  }

  return (
    <div className="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4">
      {mocs.map((moc) => (
        <MocCard key={moc.id} moc={moc} />
      ))}
    </div>
  )
}
```
