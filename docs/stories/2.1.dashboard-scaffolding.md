# Story 2.1: Dashboard Project Scaffolding

## Status

Approved

## Story

**As a** developer,
**I want** the dashboard-app project scaffolded,
**so that** I can build the dashboard features.

## Acceptance Criteria

1. ⬜ `apps/web/dashboard-app/` folder created
2. ⬜ package.json with correct dependencies
3. ⬜ Connected to shell via lazy loading
4. ⬜ Route `/dashboard` configured in shell
5. ⬜ Basic page component renders

## Tasks / Subtasks

- [ ] **Task 1: Create App Structure**
  - [ ] Create folder `apps/web/dashboard-app/`
  - [ ] Create package.json with name `@repo/dashboard-app`
  - [ ] Add dependencies: react, @repo/ui, @reduxjs/toolkit

- [ ] **Task 2: Shell Integration**
  - [ ] Create DashboardModule.tsx in shell
  - [ ] Configure lazy loading
  - [ ] Add /dashboard route

- [ ] **Task 3: Basic Page**
  - [ ] Create DashboardPage.tsx
  - [ ] Export from index.ts
  - [ ] Verify renders in browser
