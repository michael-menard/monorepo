# Story 3.14: No Results State

## Status

## Story

**As a** user,
**I want** a helpful message when search returns nothing,
**so that** I know to try different filters.

## Acceptance Criteria

1. ⬜ NoResults component created
2. ⬜ Displays when search/filter returns 0 items
3. ⬜ Shows current search term
4. ⬜ Suggests adjusting filters
5. ⬜ Clear filters button

## Dev Notes

```typescript
interface NoResultsProps {
  searchTerm?: string
  onClearFilters: () => void
}

export function NoResults({ searchTerm, onClearFilters }: NoResultsProps) {
  return (
    <div className="flex flex-col items-center justify-center py-16 text-center">
      <SearchX className="h-12 w-12 text-muted-foreground mb-4" />
      <h3 className="text-lg font-semibold mb-2">No MOCs found</h3>
      {searchTerm && (
        <p className="text-muted-foreground mb-4">
          No results for "{searchTerm}"
        </p>
      )}
      <p className="text-sm text-muted-foreground mb-4">
        Try adjusting your search or filters
      </p>
      <Button variant="outline" onClick={onClearFilters}>
        Clear Filters
      </Button>
    </div>
  )
}
```
