# Story 5.13: Add Item Modal

## Status

Approved

## Story

**As a** user,
**I want** a modal to add wishlist items,
**so that** I can quickly enter details.

## Acceptance Criteria

1. ⬜ Modal dialog opens
2. ⬜ Contains add item form
3. ⬜ Close button (X)
4. ⬜ Cancel button
5. ⬜ Closes on successful save

## Tasks / Subtasks

- [ ] **Task 1: Create Modal**
  - [ ] Use Dialog from @repo/ui
  - [ ] Form content

- [ ] **Task 2: Modal Behavior**
  - [ ] Controlled open state
  - [ ] Close handlers

## Dev Notes

```typescript
interface AddItemModalProps {
  open: boolean
  onOpenChange: (open: boolean) => void
}

export function AddItemModal({ open, onOpenChange }: AddItemModalProps) {
  const handleSuccess = () => {
    onOpenChange(false)
  }

  return (
    <Dialog open={open} onOpenChange={onOpenChange}>
      <DialogContent className="sm:max-w-md">
        <DialogHeader>
          <DialogTitle>Add to Wishlist</DialogTitle>
          <DialogDescription>
            Add a MOC, LEGO set, or alt brick set you want.
          </DialogDescription>
        </DialogHeader>
        <WishlistItemForm onSuccess={handleSuccess} onCancel={() => onOpenChange(false)} />
      </DialogContent>
    </Dialog>
  )
}
```
