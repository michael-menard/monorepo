# Story 5.20: External Link Handling

## Status

Approved

## Story

**As a** user,
**I want** external links to open securely,
**so that** I can visit item pages safely.

## Acceptance Criteria

1. ⬜ Links open in new tab
2. ⬜ rel="noopener noreferrer" added
3. ⬜ External link icon indicator
4. ⬜ Visual distinction from internal links
5. ⬜ Accessible link text

## Dev Notes

```typescript
interface ExternalLinkProps {
  href: string
  children: React.ReactNode
  className?: string
}

export function ExternalLink({ href, children, className }: ExternalLinkProps) {
  return (
    <a
      href={href}
      target="_blank"
      rel="noopener noreferrer"
      className={cn(
        'inline-flex items-center gap-1 text-primary hover:underline',
        className
      )}
    >
      {children}
      <ExternalLinkIcon className="h-3 w-3" />
      <span className="sr-only">(opens in new tab)</span>
    </a>
  )
}

// Usage in WishlistItemCard
<ExternalLink href={item.url}>
  Visit link
</ExternalLink>
```
