# Story 1.3: ESLint & Prettier

## Status

Approved

## Story

**As a** developer,
**I want** ESLint and Prettier configured per coding standards,
**so that** code style is consistent and errors are caught early.

## Acceptance Criteria

1. ⬜ ESLint 9.x configured with Airbnb style guide
2. ⬜ Prettier configured for formatting
3. ⬜ No semicolons, single quotes, trailing commas enforced
4. ⬜ 2-space indentation
5. ⬜ TypeScript strict rules enabled
6. ⬜ React hooks lint rules enabled
7. ⬜ `pnpm lint` command works without errors
8. ⬜ `pnpm lint:fix` auto-fixes issues
9. ⬜ Import order rules match coding standards

## Tasks / Subtasks

- [ ] **Task 1: Verify ESLint Configuration** (AC: 1, 5-6)
  - [ ] Check ESLint 9.x is installed
  - [ ] Verify TypeScript parser configured
  - [ ] Confirm React/hooks plugins enabled
  - [ ] Check Airbnb style extends

- [ ] **Task 2: Verify Prettier Configuration** (AC: 2-4)
  - [ ] Check Prettier is installed
  - [ ] Verify .prettierrc or config has:
    - semi: false
    - singleQuote: true
    - trailingComma: 'all'
    - tabWidth: 2

- [ ] **Task 3: Verify Import Order** (AC: 9)
  - [ ] Check import-order plugin configured
  - [ ] Groups: external → @repo/\* → relative
  - [ ] Alphabetical within groups

- [ ] **Task 4: Test Lint Commands** (AC: 7-8)
  - [ ] Run `pnpm lint` - verify no blocking errors
  - [ ] Run `pnpm lint:fix` - verify auto-fix works
  - [ ] Document any existing lint errors to fix

## Dev Notes

### Coding Standards Reference

[Source: docs/architecture/coding-standards.md#code-style]

**Required Rules:**

- No semicolons
- Single quotes
- Trailing commas
- 2-space indentation
- No `any` types
- No `console.log` (use @repo/logger)

### Import Order

[Source: docs/architecture/coding-standards.md#import-order]

```typescript
// 1. External dependencies
import { useState } from 'react'
import { z } from 'zod'

// 2. Internal packages
import { Button } from '@repo/ui'

// 3. Relative imports
import { formatDate } from './utils'
```

### Existing Scripts

**File:** `apps/web/main-app/package.json`

```json
{
  "scripts": {
    "lint": "eslint src/ --fix",
    "format": "prettier --write src/",
    "check": "prettier --write src/ && eslint --fix src/"
  }
}
```

### Testing

- Run `pnpm lint` and capture output
- Create a test file with violations and verify they're caught
- Verify pre-commit hooks run lint
