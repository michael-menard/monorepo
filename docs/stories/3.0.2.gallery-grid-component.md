Draft

## Story

**As a** developer,
**I want** a responsive gallery grid component,
**so that** all galleries display images in a consistent, responsive layout.

## Acceptance Criteria

1. ⬜ GalleryGrid component created in @repo/gallery
2. ⬜ Responsive columns: 1 (mobile) / 2 (tablet) / 3 (desktop) / 4 (large)
3. ⬜ Accepts children or render prop for flexible card rendering
4. ⬜ Configurable gap spacing
5. ⬜ Supports CSS Grid with proper aspect ratio handling
6. ⬜ Accessible container with appropriate ARIA attributes

## Dev Notes

### Component API

```typescript
interface GalleryGridProps {
  children: React.ReactNode
  columns?: {
    sm?: number  // default: 1
    md?: number  // default: 2
    lg?: number  // default: 3
    xl?: number  // default: 4
  }
  gap?: number  // Tailwind gap value, default: 4
  className?: string
}

// Usage
<GalleryGrid>
  {items.map(item => (
    <GalleryCard key={item.id} {...item} />
  ))}
</GalleryGrid>
```

### Responsive Grid Classes

```typescript
const gridClasses = cn(
  'grid',
  'grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4',
  `gap-${gap}`,
  className
)
```
