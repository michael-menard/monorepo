# Story 4.9: MOC Form - Basic Fields

## Status

Approved

## Story

**As a** user,
**I want** to enter MOC title and description,
**so that** I can identify my MOCs.

## Acceptance Criteria

1. ⬜ Title input (required, max 100 chars)
2. ⬜ Description textarea (optional, max 500 chars)
3. ⬜ Zod validation
4. ⬜ Error messages displayed
5. ⬜ Character count shown

## Tasks / Subtasks

- [ ] **Task 1: Create Form Schema**
  - [ ] Define Zod schema
  - [ ] Title and description rules

- [ ] **Task 2: Form Fields**
  - [ ] Title Input with label
  - [ ] Description Textarea
  - [ ] Error display

## Dev Notes

```typescript
const mocFormSchema = z.object({
  title: z.string().min(1, 'Title is required').max(100, 'Max 100 characters'),
  description: z.string().max(500, 'Max 500 characters').optional(),
  // Additional fields in other stories
})

function MocFormBasicFields({ form }: { form: UseFormReturn<MocFormValues> }) {
  return (
    <>
      <FormField
        control={form.control}
        name="title"
        render={({ field }) => (
          <FormItem>
            <FormLabel>Title *</FormLabel>
            <FormControl>
              <Input {...field} placeholder="My awesome MOC" />
            </FormControl>
            <FormMessage />
          </FormItem>
        )}
      />
      <FormField
        control={form.control}
        name="description"
        render={({ field }) => (
          <FormItem>
            <FormLabel>Description</FormLabel>
            <FormControl>
              <Textarea {...field} placeholder="Describe your MOC..." rows={4} />
            </FormControl>
            <FormDescription>
              {field.value?.length || 0}/500 characters
            </FormDescription>
            <FormMessage />
          </FormItem>
        )}
      />
    </>
  )
}
```
