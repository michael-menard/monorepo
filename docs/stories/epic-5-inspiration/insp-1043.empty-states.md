# Story insp-1043: Empty States (4 scenarios)

## Status

Draft

## Story

**As a** user,
**I want** helpful empty states throughout the gallery,
**so that** I understand what to do when there's no content.

## Acceptance Criteria

1. Empty gallery (new user): welcoming illustration + "Start your collection" CTA
2. Empty gallery (all deleted): neutral message + upload CTA
3. Empty album: drop zone highlight + upload prompt
4. No search results: filter feedback + clear filters button
5. All empty states include actionable CTAs
6. Consistent styling across all states

### Reusable Component

```tsx
interface EmptyStateProps {
  icon: LucideIcon
  title: string
  description: string
  action?: {
    label: string
    onClick: () => void
    variant?: 'default' | 'outline'
  }
  secondaryAction?: {
    label: string
    onClick: () => void
  }
}

function EmptyState({
  icon: Icon,
  title,
  description,
  action,
  secondaryAction,
}: EmptyStateProps) {
  return (
    <div className="flex flex-col items-center justify-center py-12 text-center">
      <div className="w-16 h-16 rounded-full bg-muted flex items-center justify-center mb-4">
        <Icon className="w-8 h-8 text-muted-foreground" />
      </div>
      <h3 className="text-lg font-medium mb-2">{title}</h3>
      <p className="text-muted-foreground max-w-sm mb-6">{description}</p>
      {action && (
        <Button onClick={action.onClick} variant={action.variant}>
          {action.label}
        </Button>
      )}
      {secondaryAction && (
        <Button
          variant="link"
          onClick={secondaryAction.onClick}
          className="mt-2"
        >
          {secondaryAction.label}
        </Button>
      )}
    </div>
  )
}
```

### Usage Examples

```tsx
// New user
<EmptyState
  icon={Sparkles}
  title="Start your inspiration collection"
  description="Upload images of LEGO builds you'd like to create someday."
  action={{
    label: "Upload your first image",
    onClick: openUploadModal,
  }}
/>

// Empty album
<EmptyState
  icon={FolderOpen}
  title="This album is empty"
  description="Drag images here or click to upload."
  action={{
    label: "Add images",
    onClick: openUploadModal,
  }}
/>

// No search results
<EmptyState
  icon={SearchX}
  title="No inspirations match your filters"
  description="Try adjusting your search or clearing filters."
  action={{
    label: "Clear filters",
    onClick: clearFilters,
    variant: "outline",
  }}
/>
```
