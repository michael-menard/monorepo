# Story insp-1006: Inspiration Card Component

## Status

Draft

## Story

**As a** user,
**I want** to see inspiration images displayed as cards in the gallery,
**so that** I can visually browse my collection.

## Acceptance Criteria

1. InspirationCard component displays image thumbnail
2. Shows title (or "Untitled" fallback)
3. Shows tag chips (max 3 visible, +N indicator)
4. Click navigates to detail view
5. Visual distinction from album cards
6. Loading skeleton state for image
7. Lazy loads images on scroll

### Component Location

### Component Props

```typescript
interface InspirationCardProps {
  inspiration: InspirationSummary
  onClick?: () => void
}
```

### Card Design

```tsx
function InspirationCard({ inspiration }: InspirationCardProps) {
  const navigate = useNavigate()

  return (
    <Card
      className="group cursor-pointer overflow-hidden"
      onClick={() => navigate({ to: '/inspiration/$id', params: { id: inspiration.id } })}
    >
      <div className="aspect-square relative">
        <img
          src={inspiration.imageUrl}
          alt={inspiration.title || 'Inspiration image'}
          className="object-cover w-full h-full"
          loading="lazy"
        />
      </div>
      <CardContent className="p-3">
        <h3 className="font-medium truncate">
          {inspiration.title || 'Untitled'}
        </h3>
        {inspiration.tags.length > 0 && (
          <div className="flex gap-1 mt-2 flex-wrap">
            {inspiration.tags.slice(0, 3).map((tag) => (
              <Badge key={tag} variant="secondary" className="text-xs">
                {tag}
              </Badge>
            ))}
            {inspiration.tags.length > 3 && (
              <Badge variant="outline" className="text-xs">
                +{inspiration.tags.length - 3}
              </Badge>
            )}
          </div>
        )}
      </CardContent>
    </Card>
  )
}
```

### Lazy Loading
Use native `loading="lazy"` attribute or Intersection Observer for images.
