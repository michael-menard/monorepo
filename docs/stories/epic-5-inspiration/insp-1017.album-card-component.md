# Story insp-1017: Album Card Component

## Status

Draft

## Story

**As a** user,
**I want** to see albums displayed as distinct cards in the gallery,
**so that** I can visually distinguish albums from individual images.

## Acceptance Criteria

1. AlbumCard component displays cover image or placeholder
2. Shows stacked/folder visual style to distinguish from images
3. Shows album title
4. Shows item count badge
5. Click navigates into album view
6. Visual hover state indicates interactivity

### Visual Design
Per PRD: "Visual distinction between inspirations (single image) and albums (stacked/folder icon)"

Options:
1. Stacked appearance (offset shadows suggesting multiple images)
2. Folder icon overlay in corner
3. Different border/frame style

### Component Structure

```tsx
interface AlbumCardProps {
  album: AlbumSummary
}

function AlbumCard({ album }: AlbumCardProps) {
  const navigate = useNavigate()

  return (
    <Card
      className="group cursor-pointer overflow-hidden relative"
      onClick={() => navigate({ to: '/inspiration/albums/$id', params: { id: album.id } })}
    >
      {/* Stacked effect - shadow layers */}
      <div className="absolute -bottom-1 -right-1 w-full h-full bg-muted rounded-lg -z-10" />
      <div className="absolute -bottom-2 -right-2 w-full h-full bg-muted/50 rounded-lg -z-20" />

      {/* Main card content */}
      <div className="aspect-square relative bg-muted">
        {album.coverImageUrl ? (
          <img
            src={album.coverImageUrl}
            alt={album.title}
            className="object-cover w-full h-full"
          />
        ) : (
          <div className="flex items-center justify-center h-full">
            <FolderIcon className="w-12 h-12 text-muted-foreground" />
          </div>
        )}

        {/* Item count badge */}
        <Badge className="absolute bottom-2 right-2">
          {album.itemCount} {album.itemCount === 1 ? 'item' : 'items'}
        </Badge>
      </div>

      <CardContent className="p-3">
        <h3 className="font-medium truncate">{album.title}</h3>
      </CardContent>
    </Card>
  )
}
```
