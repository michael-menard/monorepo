## Dev Notes

### Existing Implementation

**File:** `apps/web/main-app/src/components/Layout/Footer.tsx`

### Component Structure

```typescript
export function Footer() {
  const year = new Date().getFullYear()
  const version = import.meta.env.VITE_APP_VERSION || '0.0.1'

  return (
    <footer className="border-t py-4 px-6">
      <div className="flex flex-col md:flex-row justify-between items-center gap-2">
        <p className="text-sm text-muted-foreground">
          © {year} LEGO MOC Instructions. All rights reserved.
        </p>
        <div className="flex gap-4 text-sm text-muted-foreground">
          <a href="/terms" className="hover:underline">Terms</a>
          <a href="/privacy" className="hover:underline">Privacy</a>
          <span>v{version}</span>
        </div>
      </div>
    </footer>
  )
}
```

### Version Source Options

1. `import.meta.env.VITE_APP_VERSION`
2. Import from package.json (requires vite config)
3. Hardcoded (not recommended)
