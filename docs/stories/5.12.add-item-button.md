# Story 5.12: Add Item Button

## Status

Approved

## Story

**As a** user,
**I want** a prominent add button,
**so that** I can quickly add wishlist items.

## Acceptance Criteria

1. ⬜ "Add Item" button in header
2. ⬜ Plus icon with label
3. ⬜ Opens add modal on click
4. ⬜ Primary button style
5. ⬜ Mobile: floating action button option

## Tasks / Subtasks

- [ ] **Task 1: Header Button**
  - [ ] Add to page header
  - [ ] Click opens modal

- [ ] **Task 2: Mobile FAB (optional)**
  - [ ] Fixed position on mobile
  - [ ] Same action

## Dev Notes

```typescript
function WishlistHeader({ onAddClick }: { onAddClick: () => void }) {
  return (
    <div className="flex items-center justify-between mb-6">
      <h1 className="text-2xl font-bold">Wishlist</h1>
      <Button onClick={onAddClick}>
        <Plus className="mr-2 h-4 w-4" />
        Add Item
      </Button>
    </div>
  )
}

// Optional FAB for mobile
function AddItemFab({ onClick }: { onClick: () => void }) {
  return (
    <Button
      onClick={onClick}
      className="fixed bottom-6 right-6 h-14 w-14 rounded-full shadow-lg md:hidden"
    >
      <Plus className="h-6 w-6" />
    </Button>
  )
}
```
