# Story 3.13: Clear Filters

## Status

Approved

## Story

**As a** user,
**I want** to clear all filters at once,
**so that** I can quickly reset the view.

## Acceptance Criteria

1. ⬜ "Clear Filters" button visible when filters active
2. ⬜ Clears search, tags, theme
3. ⬜ Resets page to 1
4. ⬜ Hidden when no filters active
5. ⬜ Updates URL

## Tasks / Subtasks

- [ ] **Task 1: Create Button**
  - [ ] Show only when filters active
  - [ ] Clear action

- [ ] **Task 2: Check Active Filters**
  - [ ] Compute hasActiveFilters

## Dev Notes

```typescript
interface ClearFiltersProps {
  onClear: () => void
  hasActiveFilters: boolean
}

export function ClearFilters({ onClear, hasActiveFilters }: ClearFiltersProps) {
  if (!hasActiveFilters) return null

  return (
    <Button variant="ghost" size="sm" onClick={onClear}>
      <X className="mr-2 h-4 w-4" />
      Clear Filters
    </Button>
  )
}

// Usage
const hasActiveFilters = q || tags.length > 0 || theme
const handleClear = () => {
  navigate({ search: { q: '', tags: [], theme: null, page: 1 } })
}
```
