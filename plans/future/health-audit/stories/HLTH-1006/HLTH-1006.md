# HLTH-1006: Add missing peer dependencies to @repo/cache

## Context

`packages/core/cache/src/components/CacheDashboard/index.tsx` imports `@repo/app-component-library` and `lucide-react` but neither is declared as a dependency in `@repo/cache`'s `package.json`. TypeScript reports TS2307 (`Cannot find module`) for both.

## Goal

Add the two missing dependencies to `@repo/cache`'s `package.json` so the package compiles cleanly.

## Non-Goals

- No changes to CacheDashboard component functionality

## Scope

```bash
cd packages/core/cache
pnpm add @repo/app-component-library@workspace:* lucide-react
```

## Acceptance Criteria

### AC1: Type errors cleared
**When** `pnpm turbo check-types --filter=@repo/cache` runs
**Then** zero TS2307 errors

## Test Plan

- `pnpm turbo check-types --filter=@repo/cache`
- `pnpm turbo build --filter=@repo/cache`
