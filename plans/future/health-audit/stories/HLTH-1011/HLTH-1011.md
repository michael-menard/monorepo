# HLTH-1011: Install msw dev dependency in @repo/api-client

## Context

`@repo/api-client` test files import from `msw/node` but `msw` is not listed as a dev dependency in `packages/core/api-client/package.json`. This causes 3 test files to fail at setup with `Cannot find module 'msw/node'`.

## Goal

Add `msw` as a dev dependency in `@repo/api-client` so test files can use MSW for request mocking.

## Non-Goals

- Not adding MSW handlers or tests — only making the package available

## Scope

```bash
cd packages/core/api-client
pnpm add -D msw
```

If MSW is already at the workspace root, use `msw@workspace:*` or `msw@*` to reference it.

## Acceptance Criteria

### AC1: msw resolves in api-client tests
**When** `pnpm turbo test --filter=@repo/api-client` runs
**Then** no `Cannot find module 'msw/node'` errors

## Test Plan

- `pnpm turbo test --filter=@repo/api-client`
