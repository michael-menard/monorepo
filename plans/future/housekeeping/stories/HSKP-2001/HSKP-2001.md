# HSKP-2001: Express Local Development Server

## Context

Currently, running the API locally requires either deploying to AWS (slow, expensive) or using `serverless-offline` (heavy, complex, has parity gaps with real Lambda). With HSKP-2000 extracting business logic into transport-agnostic handlers, it becomes straightforward to mount those handlers on an Express server — providing a fast, simple local development environment with no AWS dependency.

This story builds on the hexagonal architecture introduced in HSKP-2000. The portable handlers from `@repo/api-handlers` are adapted to Express via thin Express-to-web-Request/Response adapters.

---

## Goal

Provide a local development environment where developers can run the full API against a local PostgreSQL database using Docker Compose, without any AWS connectivity. Setup time from clone should be under 5 minutes.

---

## Non-Goals

- Production deployment via Express (Lambda remains the production runtime)
- Replacing `serverless-offline` for integration tests that specifically test Lambda behavior
- Implementing a local Cognito mock (Cognito JWT middleware will validate against real Cognito in dev, or a configurable bypass for purely local work)
- Hot-reloading of database schema (Drizzle migrations must be run manually)

---

## Scope

### Docker Compose
- `docker/docker-compose.local.yml` with PostgreSQL 16 service
- Volume for data persistence across container restarts
- `.env.local.example` with all required environment variables documented

### Express Application (`apps/api-express`)
- New app in the monorepo: `apps/api-express`
- Express server that mounts all handlers from `@repo/api-handlers`
- Express-to-web-standard adapter: converts `express.Request` → `Request`, `Response` → `express.Response`
- Cognito JWT middleware (validates JWT against Cognito public keys, or accepts a `LOCAL_AUTH_BYPASS=true` env flag for purely local work without Cognito)
- Structured logging via `@repo/logger`

### Developer Workflow
```bash
# Start local DB
docker compose -f docker/docker-compose.local.yml up -d

# Run migrations
pnpm --filter @repo/api db:migrate

# Start local API + frontend together
pnpm dev:local
```

### pnpm Scripts
- `pnpm dev:local` added to root — starts both `apps/api-express` and the frontend in parallel via Turborepo
- `apps/api-express` has its own `dev` script using `tsx watch`

- [ ] **AC2**: `docker compose -f docker/docker-compose.local.yml up -d` succeeds and the DB is reachable within 30 seconds
- [ ] **AC3**: `.env.local.example` documents all environment variables required to run the Express server

### Express App
- [ ] **AC4**: `apps/api-express` package exists and starts without errors via `pnpm dev:local`
- [ ] **AC5**: All routes registered in Express correspond 1:1 with the Lambda handler routes in `serverless.yml`
- [ ] **AC6**: Express adapter correctly translates path parameters, query strings, and request body onto the web-standard `Request` object
- [ ] **AC7**: Express adapter correctly serialises the handler's `Response` back to the Express response (status, headers, body)

### Authentication Middleware
- [ ] **AC8**: Requests with a valid Cognito JWT pass through to the handler with `userId` populated in `HandlerContext`
- [ ] **AC9**: Requests with an invalid or missing JWT return 401
- [ ] **AC10**: When `LOCAL_AUTH_BYPASS=true` is set, requests without a JWT are accepted and a configured `LOCAL_USER_ID` is injected into `HandlerContext`

- [ ] **AC14**: Setup from a clean clone to a running local API takes under 5 minutes following the README instructions

## Implementation Notes

- Use `tsx` (not `ts-node`) for local development server — faster startup, no compilation step
- The Express adapter is thin: it should not contain business logic, only protocol translation
- Port number defaults to 3001 to avoid conflict with the frontend dev server on 3000 (or 5173 for Vite)
- The JWT bypass (`LOCAL_AUTH_BYPASS`) must be explicitly disabled in any CI or staging environment — add a safeguard that throws if `NODE_ENV=production` and the flag is set
