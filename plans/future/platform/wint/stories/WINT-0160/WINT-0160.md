This story validates two operating modes:

**Mode 1 — File-Only (database unavailable):**
- Run `/doc-sync` without database access
- Verify SYNC-REPORT.md shows `database_status: unavailable`
- Verify documentation files updated from file frontmatter
- Verify no crash or unhandled exception
