ARCHITECTURE CHANGE
================================================================================

OLD: public (KB items) + workflow (everything else) + drop 5 empty schemas
NEW: public (graph — all traversable data) + analytics (append-only telemetry)

Key changes:
  - No workflow schema — graph traversal shouldn't cross schema boundaries
  - analytics schema for telemetry only (story_token_usage, model_experiments, etc.)
  - New graph infrastructure: story_knowledge_links edge table, story embeddings
  - New tools: kb_find_similar_stories, kb_get_story_context
  - Phase 0 added for migration infrastructure (was missing)
  - Phase 2 redesigned from "operations tables" to "graph infrastructure"
  - CDTS-3030 collapsed into CDTS-3020
