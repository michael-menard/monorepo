# lnch-1013: Lambda Troubleshooting Runbook

## Context

Phase 2 (Before soft launch).

Lambda functions are the primary compute layer for the API. Cold starts, memory limits, timeout errors, and permission denials are the most common failure modes in production serverless environments. On-call engineers need a systematic troubleshooting guide that walks through the diagnostic steps from first alert to resolution. This runbook is referenced by the on-call playbook (lnch-1024).

## Goal

Write a Lambda troubleshooting runbook at `docs/operations/runbooks/lambda-troubleshooting.md` covering the most common Lambda failure scenarios with step-by-step diagnosis and resolution procedures.

## Non-Goals

- Do not document Lambda deployment (lnch-1009).
- Do not document database troubleshooting (lnch-1015).
- Do not document general CloudWatch usage (lnch-1029).

## Scope

**Output file:** `docs/operations/runbooks/lambda-troubleshooting.md`

**Content outline:**

1. **Overview** — Lambda functions in this project, how to identify which function is failing
2. **Common failure modes**
   - Timeout errors — symptoms, causes, diagnosis steps, resolution
   - Memory limit exceeded — symptoms, causes (memory setting vs actual usage), resolution
   - Cold start latency spikes — when this matters, mitigation options (provisioned concurrency)
   - Permission denied (IAM) — how to read the error, finding the missing permission, fix
   - VPC connectivity issues — Lambda-to-RDS, Lambda-to-OpenSearch failures
   - Unhandled promise rejections — how they surface in logs, finding the root cause
3. **CloudWatch Logs navigation** — log group naming convention, useful filter patterns
4. **X-Ray trace analysis** — finding the slow segment in a trace
5. **Concurrency and throttling** — how to spot throttling, how to request limit increase
6. **Environment variable issues** — verifying env vars are set correctly in the function config
7. **Escalation criteria** — when to escalate to Aurora DBA (lnch-1020), VPC team (lnch-1019)

## Acceptance Criteria
