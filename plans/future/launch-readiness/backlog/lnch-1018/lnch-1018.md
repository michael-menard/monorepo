# lnch-1018: Module Federation Debugging Runbook

## Context

Phase 3 (Before general availability).

The frontend uses Module Federation to compose the shell app and remote modules (e.g., app-dashboard as a remote). Module Federation failures are notoriously difficult to debug because errors are often runtime exceptions with cryptic messages, and the failure mode depends on the deployment order, remote entry URL availability, and version compatibility. This runbook bridges the gap for engineers unfamiliar with Module Federation internals.

## Goal

Write a Module Federation debugging runbook at `docs/operations/runbooks/module-federation.md` covering the most common failure modes and their diagnosis/resolution steps.

## Non-Goals

- Do not redesign the Module Federation architecture.
- Do not document general frontend deployment (lnch-1011).
- Do not document Webpack/Vite configuration changes.

## Scope

**Output file:** `docs/operations/runbooks/module-federation.md`

**Content outline:**

1. **Architecture overview** — shell app, remote modules, remoteEntry.js files, shared dependencies
2. **Common failure modes**
   - Remote entry URL not found (404) — symptoms, cause (deploy order, CDN miss), resolution
   - Shared dependency version mismatch — symptoms (React hook errors, duplicate React), diagnosis, resolution
   - Remote module load timeout — symptoms, diagnosis (network tab), resolution
   - Stale remoteEntry.js cached by CloudFront — symptoms, cache invalidation fix
   - Container not initialized errors — cause, resolution
3. **Browser devtools diagnostic steps** — Network tab, Console errors, what to look for
4. **Local debugging setup** — how to run shell + remote locally to reproduce issues
5. **Deployment order verification** — how to confirm remotes are deployed before shell
6. **Version compatibility matrix** — how to check which shell version works with which remote versions
7. **Escalation** — when to engage the frontend architecture owner

## Acceptance Criteria

- [ ] Runbook exists at `docs/operations/runbooks/module-federation.md`

- [ ] Browser devtools steps include specific console error strings to search for
- [ ] Local reproduction setup is documented
