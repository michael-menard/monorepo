# LEGO MOC Instructions Design System

Welcome to the Storybook for the LEGO MOC Instructions platform. This is the single source of truth for our component library.

## Overview

Our design system is built with:

- **React 19** - Latest React features
- **Tailwind CSS v4** - Utility-first CSS
- **Radix UI** - Accessible component primitives
- **Framer Motion** - Smooth animations
- **shadcn/ui** - Beautiful, accessible components

## Color Palette

Our theme uses a LEGO-inspired Sky/Teal color palette:

| Color | Usage |
|-------|-------|
| Sky | Primary actions, links, focus states |
| Teal | Secondary actions, accents |

## Getting Started

Browse the sidebar to explore our components. Each component includes:

- **Docs** - Usage documentation and props
- **Controls** - Interactive playground
- **A11y** - Accessibility audit results

## Import Convention

Always import from `@repo/ui`:

```tsx
import { Button, Card, Table } from '@repo/ui'
```

Never import from individual component paths.
