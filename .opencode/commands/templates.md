Use the GitHub templates in this project:

## Issue Templates

Located in `.github/ISSUE_TEMPLATE/`:

- `feature_request.md` - New features → becomes story
- `bug_report.md` - Bugs with reproduction steps → becomes bug story
- `question.md` - Questions
- `epic_proposal.md` - Strategic initiatives → becomes plan

## Pull Request Template

Located in `.github/PULL_REQUEST_TEMPLATE/PULL_REQUEST_TEMPLATE.md`

- Auto-populates PR body

## PR Review Checklist

Located in `.github/pull_request_review_template.md`

- Copy/paste into PR comments

## Commit Template

Located in `.github/commit_template.txt`

- Already configured for `git commit`

Run `git commit` to see the commit template. Use `/github-templates` skill for details.
