# /wt-status - Worktree Status

Show comprehensive status of git worktrees and database-tracked story associations.

## Usage

```
/wt-status
```

## Shows

- All active git worktrees
- Their branches and status
- Story associations from database
- Orphaned worktrees (no DB record)
- Untracked worktrees (not in git)
