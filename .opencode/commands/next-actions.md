# /next-actions - Next Actions

Query the KB database for the next unblocked stories and recommend commands.

## Usage

```
/next-actions
```

## Shows

- Next stories ready for work
- Their priority and status
- Recommended commands to proceed

## Notes

- Falls back to WORK-ORDER-BY-BATCH.md if KB has no stories
- Helps identify what to work on next
