# /roadmap - Show Roadmap

Show the active roadmap - current and future plans only (excludes implemented, superseded, archived).

## Usage

```
/roadmap
```

## Shows

- In-flight stories and their progress
- Upcoming planned stories
- Feature priorities
- Status of each plan

## Notes

- Quick view of what's in flight and coming next
- Excludes completed/archived items
